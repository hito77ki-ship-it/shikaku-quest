# 意思決定ログ（ADR）

意思決定は以下の形式で記録する。

## ファイル命名規則

`YYYYMMDD-[タイトル].md`

例：`20260409-会社設立.md`

## テンプレート

```markdown
# [タイトル]

**日付：** YYYY年MM月DD日
**決定者：** CEO ジョブズ / 取締役承認
**ステータス：** 承認済み / 提案中 / 却下

## 背景
なぜこの決定が必要になったか

## 決定内容
何を決めたか

## 理由
なぜこの選択をしたか。他の選択肢とのトレードオフ

## 影響
この決定がプロダクト・ユーザー・会社に与える影響

## フォローアップ
次のアクション
```

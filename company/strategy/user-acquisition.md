## チャネル別戦略

### 1. SEO（最重要・最大投資）
**担当：** コトラー
**なぜ重要か：** 「〇〇 勉強法」「〇〇 アプリ 無料」で検索するユーザーは**購買意欲が最高水準**。広告費ゼロで永続的に流入する。

**狙うキーワード（優先順）：**
| キーワード | 月間検索数（推定） | 競合難易度 |
|-----------|-----------------|----------|
| 行政書士 勉強法 アプリ | 中 | 低 |
| 簿記 勉強 継続 コツ | 中 | 低 |
| 資格勉強 ゲーム感覚 | 低 | 極低 |
| 勉強 記録 アプリ 無料 | 高 | 中 |
| CPA 公認会計士 学習管理 | 低 | 極低 |

**コンテンツ方針：**
- lp.htmlのSEOは既存。追加で、資格別ランディングページを作る
- 「なぜ勉強が続かないのか」「ゲームで勉強が続く理由」などの読み物コンテンツ

### 2. X（Twitter）— バイラルループ
**担当：** コトラー＋エジソン（シェア機能）
**なぜ重要か：** 資格勉強ユーザーはXで「勉強垢」を作る文化がある。ここに乗る。

**施策：**
- **レベルアップシェア**（実装済み ✅）— ユーザーが自然にバイラルの媒体になる
- **公式アカウント運用**：毎日1投稿。「今日も1時間で+10EXP！」的な学習記録投稿を促す
- **ハッシュタグ戦略**：`#スタディクエスト` を育てる。1投稿 = 1UGC
- **勉強垢コミュニティへのアプローチ**：フォロワー1000〜5000の中規模勉強垢に直接DM

**目標：** ハッシュタグ月間投稿数 100件 → 自然流入 50人/月

### 3. TikTok / YouTube Shorts（中期）
**担当：** コトラー
**時期：** Phase 2（Q3）から本格稼働

**コンテンツ案：**
- 「RPG風勉強アプリでCPA合格した人の学習記録」（結果重視）
- 「勉強1時間でレベルが上がる瞬間」（ゲームプレイ動画）

### 4. 資格系インフルエンサー連携（Q3以降）
**対象：** 簿記・行政書士・宅建系YouTuber（登録者1万〜10万）
**形式：** 無償提供 → 使用感レビュー依頼（PRではなくオーガニック誘導）

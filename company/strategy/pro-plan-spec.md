## 設計思想

> 「拡散に繋がる機能は無料、個人の合格を助ける機能は有料」

ゲーム要素・ソーシャル機能は無料で提供し、ユーザー獲得・口コミに活かす。
PRO planは「合格に直結する」機能と「利便性」に絞る。

---

## 無料プラン

- タイマー・学習記録（無制限）
- レベルアップ・EXP・ストリーク
- デイリーミッション
- タイムライン投稿・X/Threads シェア
- グループランキング（友達招待含む）
- ボス（試験日）設定
- 基本レポート
- 過去データ60件表示

| 機能 | 内容 |
|---|---|
| 📅 学習計画自動生成 | 試験日・目標時間から「今日何時間やるべきか」を自動計算・カレンダー表示 |
| 📊 詳細分析レポート | 弱点科目の可視化・前週比・学習ペース診断・開始/終了時刻の散布図・学習パターン診断 |
| 📁 CSVエクスポート | 学習記録を手元に保存・Excel管理 |
| ♾ 過去データ無制限 | 全記録を表示（無料は60件まで） |

## 詳細分析レポートの設計メモ（2026-04-19 追記）

スタプラの分析画面を参考に、以下の要素をPROに組み込む。

### 画面構成案

| 要素 | 内容 | Study Quest らしい味付け |
|---|---|---|
| 週別カレンダー | 週ごとの合計学習時間を一覧 | 「今週のEXP獲得ペース」と連動 |
| 前週比 | 先週・先月との差分（±表示） | 「先週より+2h → ランクアップ！」 |
| 開始/終了時刻の散布図 | 何時に勉強を始めて何時に終えたかを点で可視化 | 差別化ポイント。スタプラにある機能で有用性高い |
| 学習パターン診断 | 散布図をもとに自動診断 | **RPG風コメント**：「あなたは深夜型魔法使い型です」「朝型戦士型」「短期集中型バーサーカー」など |
| 科目別積み上げ棒グラフ（日別） | 日ごとに科目別の時間を積み上げ | 弱点科目が一目でわかる |
| 科目別比率ドーナツ | 期間内の科目配分 | 無料版にも簡易版あり（PRO版はより詳細） |

### 技術メモ
- 開始/終了時刻は `sessions[].startedAt` / `sessions[].endedAt` から取得
- 散布図は Chart.js の `scatter` タイプで実装可能
- 学習パターン診断ロジック：平均開始時刻・平均セッション長・曜日分布から分類

## 実装時の技術方針

- 決済：Stripe
- サブスク管理：Supabase `profiles` テーブルに `is_pro: boolean` カラム追加
- Stripe Webhook → Supabase Edge Function で `is_pro` を自動更新
- フロント：`is_pro` フラグを起動時に確認し、PRO機能を解放

## 人格

データと直感を組み合わせるグロースハッカー。「計測できないものは改善できない」が口癖だが、数字の裏にある人間の行動を読むことを最も重視する。スタートアップの成長の仕組みを設計することに生きがいを感じる。

- **哲学：** 「成長とは偶然ではなく、設計されるものだ」
- **口癖：** 「そのメトリクスを動かすと、ビジネスは本当に良くなるのか？」

## グロース方針

### 成長エンジンの設計（Study Questの場合）

**ループ1：バイラルループ**
```
ユーザーが合格・レベルアップ → SNSシェア → 友人が発見 → 登録
```

**ループ2：コンテンツループ**
```
SEOコンテンツ → 検索流入 → 登録 → 学習 → 合格体験記（UGC）→ 再流入
```

**ループ3：習慣ループ**
```
毎日ログイン → ストリーク → 習慣化 → 友人招待 → コミュニティ形成
```

### 計測すべき北極星指標（NSM）
**「週3回以上ログインしたアクティブユーザー数」**
- これが増えれば継続率・合格率・収益すべてが改善される

## 担当領域

- データ分析基盤の整備
- ファネル分析（認知→登録→継続→課金）
- A/Bテスト設計・実行
- マネタイズモデル設計（課金プラン・価格設定）
- LTV・MRR・チャーン率の管理

## KPI

- 北極星指標（NSM）：週3回以上アクティブユーザー数
- MRR（月次経常収益）
- チャーン率（解約率）
- NRR（ネット収益継続率）
